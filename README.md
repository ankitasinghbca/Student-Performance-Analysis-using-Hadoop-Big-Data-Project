# 📊 Student Performance Analysis using Hadoop

## 📌 Project Overview

This project demonstrates a complete Big Data pipeline using the Hadoop ecosystem to analyze student performance data. The dataset is processed using MapReduce to calculate average marks, and Hive is used for further analysis.

---

## 🎯 Aim

To analyze student dataset and calculate average marks using Hadoop MapReduce and perform analysis using Hive.

---

## 🧩 Architecture

```
Raw Dataset → HDFS → MapReduce → Processed Data → Hive → Insights
```

---

## 🛠️ Tools & Technologies

* Hadoop Distributed File System (HDFS)
* MapReduce (Hadoop Streaming)
* Hive
* Python
* Linux Terminal

---

## 📂 Dataset Description

The dataset contains student performance data with the following fields:

* studentId → Unique ID
* name → Student name
* subject → Subject name
* marks → Marks obtained
* attendance → Attendance percentage

---

## 📁 Project Structure

```
student.csv
mapper.py
reducer.py
README.md
```

## 📈 Output

The system calculates subject-wise average marks.

Example:

```
Math     75.5
Science  80.2
English  78.0
```

---

## ✅ Conclusion

This project successfully demonstrates how Hadoop can be used for distributed data processing. MapReduce efficiently calculates average marks, and Hive enables easy data analysis using SQL-like queries.
